<Section icon={Sparkles} title="Rarezas y Selección de Mutaciones">

Cada tirada en ¡Rueda Anime para luchar! puede otorgar una unidad con estadísticas, rasgos y mutaciones únicos. Las unidades mutadas como Oro, Diamante, Nen y Titán proporcionan mejoras de rendimiento importantes que pueden superar a las unidades estándar de una rareza base superior.

| Rareza / Mutación | Prioridad de Conservación | Mejor Uso | Regla de Fusión |
|---|---|---|---|
| **Mítico** | **Máxima** | Dañador principal del equipo | Guardar todas las copias para fusiones de alto nivel |
| **Legendario** | **Alta** | Carry de progresión a mitad de juego | Fusionar copias iguales para sostener el daño |
| **Mutación Nen** | **Rango S** | Escalado de oleadas en juego tardío | Bloquear inmediatamente y crear línea de fusión Nen |
| **Mutación Titán** | **Rango S** | Escalado de daño de alto valor | Mantener separado de las líneas de fusión estándar |
| **Mutación Diamante** | **Rango A** | Mejoras de estadísticas premium | Fusionar solo con copias de Diamante iguales |
| **Mutación Oro** | **Rango B** | Progresión de inicio a mitad de juego | Usar para mejorar espacios del equipo activo |
| **Duplicado Estándar** | **Material de Fusión** | Subir nivel de unidades activas | Alimentar a tus unidades principales equipadas |

<Callout type="success" title="Consejo de Mutación">
Bloquea siempre tus unidades mutadas tan pronto como las consigas. Esto evita que uses accidentalmente una mutación de élite como simple material de nivel para una unidad estándar.
</Callout>

</Section>

<Section icon={Target} title="Cómo Fusionar y Subir de Nivel Unidades">

El sistema de fusión es el método principal para escalar el poder de tu equipo y cumplir con los requisitos de los puntos de control de fin de juego. Fusionar unidades duplicadas aumenta su nivel y estadísticas base sin requerir recursos de nivel externo.

<Steps>
  <Step num="1" title="Seleccionar la Unidad Principal">
    Compara estadísticas base, rango, enfriamientos y mutaciones para seleccionar la unidad base más fuerte que recibirá las mejoras.
  </Step>
  <Step num="2" title="Aislar Ramas Mutadas">
    Mantén separadas las variantes normales, de Oro, Diamante, Nen y Titán. No alimentes una unidad estándar con una unidad mutada.
  </Step>
  <Step num="3" title="Reunir Duplicados Iguales">
    Realiza tiradas para obtener unidades adicionales y guarda copias de los luchadores específicos que tienes equipados actualmente.
  </Step>
  <Step num="4" title="Realizar la Fusión">
    Usa la interfaz de fusión del juego para combinar duplicados iguales, aumentando el nivel de la unidad principal y su DPS general.
  </Step>
  <Step num="5" title="Evaluar Estadísticas Actualizadas">
    Compara el rendimiento de la unidad recién mejorada con el resto de tu equipo para asegurar que mantenga una alta eficiencia.
  </Step>
</Steps>

<Callout type="info" title="Estrategia de Fusión">
Una unidad Legendaria o mutada con muchas fusiones frecuentemente superará a una unidad Mítica de nivel base sin fusiones. No reemplaces tus unidades niveladas hasta que tus nuevas adquisiciones tengan las estadísticas para igualarlas.
</Callout>

</Section>
