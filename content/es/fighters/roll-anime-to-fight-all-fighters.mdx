<Section icon={Sparkles} title="Sistema de mutación y prioridad de conservación">

Las mutaciones son modificadores visuales y de estadísticas raros que pueden activarse al obtener una unidad. Una versión mutada de una unidad de nivel inferior puede superar fácilmente a una unidad estándar de alta rareza. Mantén las ramas de mutación organizadas y nunca las fusiones descuidadamente.

| Mutación | Nivel | Valor práctico | Uso recomendado |
| :--- | :--- | :--- | :--- |
| **Nen** | **S** | Mejora de daño y velocidad de élite | Bloquear inmediatamente; núcleo del equipo |
| **Titán** | **S** | Multiplicador de daño masivo | Mantener como rama de mejora dedicada |
| **Diamante** | **A** | Mejora visual y de estadísticas premium | Fusionar solo con unidades Diamante coincidentes |
| **Oro** | **B** | Mejora visual y potenciador moderado | Excelente para la progresión de mitad de juego |
| **Ninguna** | **C** | Estadísticas base estándar | Usar como material de fusión estándar |

<Steps>
  <Step num="1" title="Buscar etiquetas de mutación">
    Al obtener una unidad, busca etiquetas especiales como Oro, Diamante, Nen o Titán en la carta de la unidad.
  </Step>
  <Step num="2" title="Bloquear unidades mutadas">
    Bloquea inmediatamente tus unidades mutadas en tu inventario para evitar que se usen accidentalmente como comida para fusión.
  </Step>
  <Step num="3" title="Crear líneas de mejora separadas">
    Solo fusiona duplicados mutados coincidentes (por ejemplo, fusiona Oro con Oro) para preservar el bono de mutación de alto nivel.
  </Step>
</Steps>

<Section icon={Shield} title="Cómo fusionar y subir de nivel a las unidades">

El sistema de fusión es el método principal para escalar el poder de tu equipo y que coincida con la dificultad de los puntos de control más altos. La fusión combina unidades duplicadas para aumentar su nivel y estadísticas de combate.

| Paso de fusión | Acción requerida | Consejo de eficiencia |
| :--- | :--- | :--- |
| **1. Seleccionar base** | Elige la unidad con la mejor mutación o estadísticas | Nunca uses una unidad mutada como comida para una normal |
| **2. Reunir copias** | Colecciona duplicados idénticos al tirar | Mantén los espacios activos llenos antes de usar duplicados |
| **3. Ejecutar fusión** | Usa el menú de fusión para combinar duplicados | Concéntrate en subir de nivel una unidad carry principal primero |
| **4. Verificar estadísticas** | Revisa el DPS, alcance y enfriamiento mejorados | Vuelve a equipar la unidad mejorada en tu alineación activa |

<Callout type="info" title="Eficiencia de fusión">
Una unidad Legendaria o Épica mutada con muchas fusiones suele ser mucho más fuerte que una unidad Mítica base sin fusionar. No cambies tus unidades con nivel hasta que tus nuevas unidades de alta rareza hayan sido fusionadas adecuadamente.
</Callout>

</Section>
